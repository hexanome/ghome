SETUP
=====


Redis
-----

Firstly, please install redis. You can fetch it from
<http://redis.googlecode.com/files/redis-2.4.7.tar.gz>.

The redis system has the following commands:

- `redis-server`
- `redis-benchmark`
- `redis-cli`
- `redis-cli`
- `redis-check-dump`
- `redis-check-aof`

Please launch `redis-server` in your command prompt.
